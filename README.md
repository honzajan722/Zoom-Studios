# 🎬 Zoom Studios — Official Website

Modern, responsive, and high-performance landing page designed for **Zoom Studios** — a creative studio specializing in virtual events, video production, audio recording, and digital experiences.

![Zoom Studios Preview](https://via.placeholder.com/1200x600/0a0b10/00f2fe?text=Zoom+Studios+Web+Interface)

---

## ✨ Features

* **🎨 Modern Aesthetic:** Glassmorphism UI elements, dark mode theme, and vibrant neon gradients (`#00f2fe`, `#ff0844`, `#4facfe`).
* **📱 Fully Responsive:** Seamless experience across desktop monitors, laptops, tablets, and mobile smartphones.
* **🍔 Interactive Mobile Navigation:** Toggleable hamburger menu tailored for touch devices.
* **🚀 Lightweight & Fast:** Built with pure HTML5, CSS3, and Vanilla JavaScript — zero external dependencies or heavy frameworks.
* **🌐 Web Fonts:** Uses Google Font (*Plus Jakarta Sans*) for a clean typography scale using dynamic CSS `clamp()`.

---

## 🛠️ Tech Stack

* **HTML5** — Semantic page structure
* **CSS3** — Custom properties (variables), CSS Grid, Flexbox, media queries & glassmorphism backdrop filters
* **JavaScript (Vanilla)** — Lightweight mobile menu state toggling
